## まとめ

`2026-03-29 17:50` 実行分でエラー。原因は **arXiv API のタイムアウト + 429（レートリミット）**。

```
arXiv error (ti:agent AND cat:cs.AI): The read operation timed out
arXiv 429, retrying in 5s (ti:agentic AND cat:cs.LG)
arXiv error (ti:agentic AND cat:cs.LG): HTTP Error 429: Unknown Error
```

その後 `経営戦略・コンサル` のスキャン中に全体のタイムアウト（180秒）に達した模様。実行時間が **1032秒**（= 17分）になってるのも怪しい。

直後の `20:50` 実行は成功してるから、arXiv が一時的に詰まってただけかも。

**現状の問題点：**
1. arXiv タイムアウト (15秒) が複数カテゴリで発生 → 積み重なって全体が遅延
2. arXiv 429 → retry 5秒待ち → 更なる遅延
