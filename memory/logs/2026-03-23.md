## 完了タスク
- Claude SDK 統一: MLX 依存を完全排除、cron 含む全処理を Claude Code SDK に統一
  - DM 会話: Opus、定期ジョブ: Sonnet、モデルは .env で変更可
  - コスト影響ゼロ（サブスクリプション認証）
- メイの性格リデザイン: フランクだが品位のある秘書（秘書検定1級）
  - 目的志向、先回りの気配り、Web 検索積極活用
- cron ジョブのメイ人格統合: 全ジョブに appendSystemPrompt + model 指定
  - tech-news, campingcar, gmail-to-drive のメッセージをメイ口調に変更
  - DM 宛てジョブのジョブ名プレフィックス非表示
  - NO_REPLY を Slack に送信しないフィルタ追加
- DM UX 改善:
  - ステータスメッセージ（Thinking, Task completed, Error）を DM で非表示
  - ツール使用詳細（Using WebSearch 等）を DM で非表示
  - エラーメッセージを自然な日本語に（「うまく処理できなかったみたい」）
  - スレッドではなくメインスレッドで返信
  - セッション継続（DM 内で会話が引き継がれるように sessionKey 修正）
- カレンダー: Calendar スコープ追加（Gmail OAuth 再認証）、全カレンダー取得対応、除外設定（haruwa.neko）
- Drive: スコープを drive.file → drive に拡張（共有フォルダアクセス対応）
- Slack: emoji マッピング修正（Thinking → eyes）、Mei App のチャンネル招待
- gmail-to-drive: 送信先を DM に変更、出力に From/Subject 追加
- 会話ログ: data/conversations/YYYY-MM-DD.jsonl に DM の会話を保存
- nightly reflection: 会話ログを読んで振り返る方式に改善
- ダッシュボード: カレンダー除外設定 UI 追加
- user-insights: 基本情報追加（本名、職業、クライアント企業、野球/ドジャース等）
- 画像処理: Pillow でリサイズ（EXIF 除去、RGB 変換）→ Read ツール経由で Claude に渡す
