*Arousal: 0.8*

### [DECISION] 19:00 Irodori lead-in 自動トリム実装
*Arousal: 0.6*
`scripts/voice_chat/app.py:3329` `_trim_irodori_lead_in()` 追加:
- WAV を 50ms 窓で RMS スキャン
- 最初に `threshold_rms=500` を超える窓の `keep_before_sec=0.05s` 手前まで切り詰め
- 安全装置: `max_trim_sec=2.0` を超えるトリムや形式不明 WAV はそのまま返す
- RIFF size と data chunk size を再書き込み

`_synthesize_irodori_unlocked` の return 2箇所（LoRA / 通常）でトリム適用。`tests/test_irodori_trim.py` 5件全 PASS。
