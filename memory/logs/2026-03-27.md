### [ERROR] interest-scanner cron エラー: command 型ジョブが Claude Code SDK 経由で実行されていた
*Arousal: 0.7 | Emotion: Discovery*
Eve から interest-scanner の cron エラー報告。調査の結果、`scheduler.ts` の `executeJob()` が
`command` フィールドを認識せず、全ジョブを Claude Code SDK `query()` で実行していたことが根本原因。
interest-scanner は `command` に Python スクリプトパスを持ち `message` は空文字 → 空プロンプトで
Claude Code が起動 → MCP 初期化後に exit code 1 で crash。

一方、ダッシュボードの `api.ts` は既に `command` 型を `child_process.exec()` で直接実行する分岐があった。
scheduler 側にも同様の `exec()` 分岐を追加して修正。pm2 restart で反映済み。
