**Google Calendar API 直接** = `collect_data.py` が持ってる OAuth credentials（`~/.gmail-mcp/`）で REST API を叩く方法。

今起きてる問題:
- gcalcli が使ってた OAuth クライアントが Google 側で削除されたので、gcalcli 経由の読み書きが全滅

1. **gcalcli の認証を `~/.gmail-mcp/` の credentials に差し替える** — gcalcli MCP が復活する
2. **gcalcli を捨てて、カレンダー操作も `collect_data.py` 系の認証に統一する** — MCP サーバーを自前で書くか、スクリプト経由に...
