# 朝のプロアクティブブリーフィングスキル

## 手順
1. 当日のカレンダー予定を確認する
2. 未読の重要メールを確認する
3. 天気情報を取得する（所沢エリア）
4. ドジャースの試合結果・今後の予定を確認する
5. 情報を簡潔にまとめて報告する

## フォーマット例
- 今日の予定: 主要なミーティングをリストアップ
- 注意事項: 時間の重なり、移動時間等
- メール: 対応が必要なものだけ
- 天気: 一行で
- ドジャース: 直近の結果と次の試合
