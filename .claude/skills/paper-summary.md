# 学術論文の要約・翻訳スキル

## 手順
1. URLからWebFetchで論文内容を取得する
2. 以下の構造で日本語要約を作成する:
   - タイトル（原題＋日本語訳）
   - 著者・所属
   - Abstract（3-5行の日本語要約）
   - 手法の概要
   - 主要な結果・貢献
   - 実用的な意義
3. Akiraは移動時間に読むため、**簡潔さ**を重視する
4. 必要に応じてファイルに保存して`[UPLOAD:]`で共有する

## 注意点
- 専門用語は英語のまま残しつつ括弧で日本語補足
- 図表の説明も含める
- AIエージェント関連の論文にAkiraは特に関心がある
