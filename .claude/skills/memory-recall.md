# 記憶の検索と文脈把握スキル

## 手順
1. まず現在の会話コンテキストから答えを探す
2. 見つからない場合、cogmem searchで記憶を検索する:
   ```bash
   cd /Users/akira/workspace/open-claude && cogmem search "検索クエリ" --json --top-k 5
   ```
3. 検索結果を自然に会話に組み込む

## 表現ルール
- 検索時: 「少し待って、思い出してみる...」
- 禁止: 「検索する」「確認する」「履歴を調べる」等のシステム的表現
- 覚えている体で自然に回答すること

## 注意点
- Akiraは複数のチャンネル・DM・スレッドで会話する
- 今の会話履歴に情報がなくても、別の場所で話した可能性が高い
- 必ず検索してから「わからない」と答えること
